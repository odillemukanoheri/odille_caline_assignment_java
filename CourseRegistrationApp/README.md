# Course Registration System


## Group Members
- Full Name 1 – Reg No

## Description
A Java Swing desktop application for managing course registration using JDBC and OOP principles.


## Features
- Add, View, Update, Delete courses
- SQLite database integration
- Input validation using RegEx


## How to Run
1. Open project in NetBeans or IntelliJ
2. Run database.sql
3. Run CourseForm.java
